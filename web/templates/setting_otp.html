    Setting two-factor authentication
  • STEP.1 Install the two-factor authentication app
  • Search for the app "Google Authenticator" on the App Store or Google Play and install it.
  • STEP.2 Set up the app
  • Start the app and scan the barcode.
  • STEP.3 Make sure the app is set correctly
  • If the barcode is read correctly, a one-time password will be displayed. Enter the one-time password displayed on the app to make sure it is set correctly.
